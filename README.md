## Description

Это тестовое задание на позицию “Backend разработчик (Стажёр) на Node.js”.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
